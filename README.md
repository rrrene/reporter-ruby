reporter-ruby
=============

A Ruby library for parsing and processing data exported from the
Reporter app for iOS.



Installation
------------

Add this line to your application's Gemfile:

    gem "reporter"

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install reporter
